Nós Somos a Uija Code, o Orgulho de Ser Angolano

  • Por que começamos?

    A Uija Code começou como um projecto de universidade que tinha de ser feito, após o projecto feito eu(Eufránio Diogo) achei que ela não deveria parar por aí porque sei que o ensino de programação para pessoas é algo muito importante.

  • Qual é o nosso objectivo?

    O nosso objectivo é ajudar a fomentar o ensino de programação de forma gratuita sem a necessidade de ter vários recursos financeiro para aprender.

  • Como nos sustentamos?

    Basicamente de ínicio a Uija Code não possui uma fonte de renda fixa, tudo que a mantém é gratuito e é desta forma que a Uija Code se mantem.

  • Quem é o nosso publico alvo?

    O nosso publico alvo são majoritamente jovens que estão interessados em aprender programação especificamente querendo começar por uma linguagem fácil de aprender que nesse caso a Uija Code escolhe a linguagem Python.

  • Será que a Uija Code nunca perderá esse enfoque free?

    O enfoque free da Uija Code será mantido, porque assim foi o seu proposito desde a sua criação.
