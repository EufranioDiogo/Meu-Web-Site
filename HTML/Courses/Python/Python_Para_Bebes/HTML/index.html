Sobre

O nome desse curso pode parecer um pouco trick(enganador) porque o curso não se trata de ensinar a programar, mas sim ensinar Python, se quiser algo que lhe ensine a programar clique no botão acima.

Objectivos

Os objectivos deste curso são: introduzir para você como Python funciona, desde algumas das suas features, detalhes particulares que a linguagem tem, listas, dicionários, tuplas, compressão de listas, funções, ciclos de repetição, estruturas de decisão, módulos, arquivos, geradores, lambda, tratamento de excessões e POO(Programação Orientada a Objectos).
